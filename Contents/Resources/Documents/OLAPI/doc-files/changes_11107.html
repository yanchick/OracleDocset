Changes in 11.1.0.7

In Oracle Database 11g, Release 1 (11.1.0.7), the Oracle OLAP Java API has added the following classes, fields, methods, and property settings.

New Classes

The following classes are new additions to the Oracle OLAP Java API. The classes are listed by package.

oracle.olapi.metadata

   BatchedException
   MetadataRenameNotSupportedException

oracle.olapi.syntax
