Description of the illustration maintwiz_dataref.gif

Selecting Build Options: This screen capture of the Maintenance Wizard Data Refresh Methods page has a table that has the following columns:

  • Object Name

  • Object Type

  • Load Option

  • Sort

  • Refresh Method

  • Cube Script

The table has five rows. The values in the rows are the following:

Row 1: TIME, Dimension MV, Synchronize, Yes, Complete, null

Row 2: CHANNEL, Dimension MV, Synchronize, Yes, Complete, null

Row 3: CUSTOMER, Dimension MV, Synchronize, Yes, Complete, null

Row 4: PRODUCT, Dimension MV, Synchronize, Yes, Complete, null

Row 5: UNITS_CUBE, Cube MV, null, null, Force, SYS_DEFAULT

The page has an Automatic Order option, which is selected and Up and Down errors that are inactive because automatic ordering is selected. The page also has Help, Back, Next, Finish, and Cancel buttons.
