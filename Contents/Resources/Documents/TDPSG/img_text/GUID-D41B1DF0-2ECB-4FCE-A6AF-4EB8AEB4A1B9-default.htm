Description of the illustration audit_create.gif

Image audit_create.gif shows the Create Audit Policy : Object Actions page. From top to bottom, left to right, are the following components:

At the top is a breadcrumb for each page of the audit policy creation process. The breadcrumbs are from left to right Privileges and Roles, Component Actions, Object Actions (which is the currently displayed page), Conditions, and Review.

Following the breadcrumb is the Create Audit Policy : Object Actions label. To the right of this label are the Back button, the label Step 3 of 5, the Next button, and the Cancel button.

Following the Create Audit Policy : Object Actions label are the instructions, "Specify object specific actions to be audited by this policy. The audit can include both DDL and DML statements that were used on the object."

Following these instructions are from left to right, the Add Object and Remove Object buttons.

Following these buttons is a table with the columns Type, Action, Schema, and Name.
