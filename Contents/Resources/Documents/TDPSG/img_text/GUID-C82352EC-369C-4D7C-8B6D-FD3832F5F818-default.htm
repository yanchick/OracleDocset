Description of the illustration dv_realm.gif

Image dv_realms.gif shows the Home page of Database Vault, with Realm selected. On the left is a selection menu for each action: Realms, Command Rules, Rules, Rule Sets, Factors, Factor Types, Secure Application Roles, OLS Integration, and Database Vault Roles.

To right of this selection menu is the Realms page. From top to bottom are the following components:

  • A brief description of what realm are used for: "Database Vault realms provide the ability to create protection zones around database objects that prevent users exercising system privileges to access data. Mandator realms also prevent users exercising object privileges and object owners accessing data from their own objects."

  • A Search field so that you can search for an existing realm. To the right of the Name label is the field, followed by the Go button on the right. You can use the wildcard symbol (%) in the search string.

  • A table listing the existing realms. The columns are Name, Audit Options, Enabled, Mandatory Realm, and Last updated date. Preceding this table are the View list, the buttons Create, View, Edit, and Delete, and the Show Oracle Defined Realms check box.
