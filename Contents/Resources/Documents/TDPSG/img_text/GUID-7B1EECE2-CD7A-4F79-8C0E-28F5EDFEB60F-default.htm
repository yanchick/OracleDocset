Description of the illustration audit_enable.gif

Image audit_enable.gif shows the Enable Audit Policy dialog. From top to bottom, left to right, are the following components:

At the top is the label Enable Audit Policy.

Following this label are the instructions, "Enabling of an object-level unified audit policy takes effect immediately. For other unified audit policy types, it takes effect on subsequent user sessions."

Following the instructions is the Enable Policy for All Users check box, deselected.

Following this check box are the Add User button, Remove User button, and Except below users check box (which enables you to exclude specific users from the audit policy).

Following these buttons is a table with the columns User, Success, and Failure.
