Description of the illustration ''%s''dbua_2.png

This is a screen shot of the DBUA Select Databases page.

The Select Database page displays the source database Oracle home.

The Source Oracle Home contains a drop-down menu where you can select a Source Oracle Home if different from the default Oracle home. Under the drop-down menu is a table with information about the selected source Oracle home and any available Oracle home found by DBUA.

The next step describes the options for this page.

At the bottom of the page are Help, Back, Next, and Cancel buttons.
