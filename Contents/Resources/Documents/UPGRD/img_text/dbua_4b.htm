The Custom Scripts tab of the Upgrade Options Configuration page provides two fields: Before Upgrade and After Upgrade, with a Browse button next to each field. The top of the page contains the following text: Specify custom SQL scripts you would like to run before/and or after upgrade. The description and options are described in the next step.

At the bottom of the page are Help, Back, Next, and Cancel buttons.
