OracleCollectionType Enumeration

OracleCollectionType enumerated values specify whether or not the OracleParameter object represents a collection, and if so, specifies the collection type.

Table 6-144 lists all the OracleCollectionType enumeration values with a description of each enumerated value.

Table 6-144 OracleCollectionType Enumeration Values

Member Name Description

None

Is not a collection type

PLSQLAssociativeArray

Indicates that the collection type is a PL/SQL Associative Array (or PL/SQL Index-By Table)
