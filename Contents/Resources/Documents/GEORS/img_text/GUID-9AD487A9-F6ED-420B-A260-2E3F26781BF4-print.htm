Description of the illustration georef_polynomials.gif

This is a text description of Figure 1-6. It is the standard mathematical graphical representation for the polynomials p, q, r, and s, which are used for georeferencing. These polynomials are explained in text before and after the illustration, and in the pseudocode and its description later in this section.
