Screen shot of the Event Properties dialog box.

This dialog box has a General tab and a Details tab.

The General tab has a text box, displaying the following text: VSS-04112: Oracle VSS writer service for database instance ORA7 stopped. Additional info: Oracle VSS writer version 12.1.0.0.0 Beta.

Below the text box, the General tab displays the Log Name, Source, Event ID, Level, User, OpCode, More Information, Logged date and time, Task Category, Keywords, and Computer for an event.

There are up and down arrow buttons on the right side of the dialog box to view properties of earlier or later events.

At the bottom of the tab are buttons labeled Copy and Close.
