E.1 Oracle Multimedia DICOM Tutorial

The sample scripts in Chapter 6 are provided to help you learn to develop applications using Oracle Multimedia DICOM to upload to the database, retrieve from it, and manipulate DICOM content.

Some sample scripts were extracted from the tutorial Managing DICOM Format Data in Oracle Database 11g, and adapted for this manual. This tutorial and other examples are available from Oracle Multimedia on the Oracle Technology Network Web site.

Enter the URL for the DICOM tutorial, as listed in Table E-1.

See Chapter 6 for more information about using these sample scripts with Oracle Multimedia DICOM.

E.2 Oracle Multimedia DICOM Image Archive Demonstration

The Oracle Multimedia DICOM Image Archive Demonstration sample application in Chapter 7 demonstrates the features of Oracle Multimedia DICOM to help you build custom applications. This sample application includes two user interfaces. The interface DICOM Image Archive is for clinicians and researchers; the interface DICOM Image Archive Administration is for administrators only.

This sample application, along with other packaged applications and sample code, is available from Oracle Multimedia on the Oracle Technology Network Web site.

Enter the URL for the DICOM Image Archive Demonstration, as listed in Table E-1.

See Chapter 7 for more information about using this sample application.
