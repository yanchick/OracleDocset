Description of the illustration clin_1.gif

This is a text description of clin_1.gif.

This is a screen capture of the DICOM Image Archive interface DICOM Image Archive login page, with the role of clinician selected. This page includes three input fields: Login As, User Name, and Password. The Login As field shows the Clinician role selected. The User Name and Password fields are text input fields, which are blank in this screen capture. This page also includes a Login button, which you click after completing the other fields on this page.
