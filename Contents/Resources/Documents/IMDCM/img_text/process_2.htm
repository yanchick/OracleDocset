Description of the illustration process_2.gif

This is a text description of process_2.gif.

This is a screen capture of some relevant parts of the DICOM Image Archive Home page, which shows the image that was processed.

At the top of the page, this message is displayed: "Processing DICOM image succeeded."

The Thumbnails area shows the anonymous thumbnail image and the copy of the selected image that was processed. The selected image is numbered 8, and the processed image is numbered 9.

The Tasks area shows the selected task: Process (Rotate/Scale/Crop) the selected DICOM images.
