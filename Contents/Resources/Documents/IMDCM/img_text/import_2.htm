This is a text description of import_2.gif.

This is a screen capture of some relevant parts of the DICOM Image Archive Home page, which shows the image that was imported to the archive.

At the top of the page, this message is displayed: "1 DICOM images are imported."

The Thumbnails area shows the six thumbnail images that are installed with the sample application and the image that was imported into the archive. The images are numbered 1 through 6, and the imported image is numbered 31.

The Tasks area shows the selected task: Add DICOM images to the archive.
