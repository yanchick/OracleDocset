OCIWideCharIsAlnum()

Purpose

Tests whether a wide character is a letter or a decimal digit.

Syntax

boolean OCIWideCharIsAlnum ( void       *hndl,
                             OCIWchar   wc );

Parameters

hndl (IN/OUT)

OCI environment or user session handle to determine the character set.

wc (IN)

The wchar for testing.

Returns

TRUE or FALSE.
