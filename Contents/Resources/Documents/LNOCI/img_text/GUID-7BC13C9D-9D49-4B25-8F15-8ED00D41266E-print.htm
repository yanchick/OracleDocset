Description of the illustration lnoci017.eps

Six boxes connected by downward pointing arrows illustrate basic OCI program flow. The boxes in order of flow are labeled as follows:

  1. Create Environment

  2. Allocate Handles and Data Structures

  3. Connect to Server and Begin Session

  4. Issue SQL and Process Data

  5. Disconnect

  6. Free Handles and Data Structures
