Description of the illustration olsag019.gif

This illustration depicts a distributed database using Oracle Label Security. At the WESTERN_REGION database, the installed Oracle Label Security policy is HR. At the EASTERN_REGION database, the installed Oracle Label Security policies are HR and DEFENSE. The WESTERN_REGION and EASTERN_REGION are subgroups of the group HQ. HQ has no Oracle Label Security policies installed. Clients connect to the various servers using Oracle Net and TCP/IP, as do the servers themselves, when they communicate with each other.
