Changes in This Release for Oracle Database Data Cartridge Developer's Guide

New Features in Oracle Database Data Cartridge 12c Release 1 (12.1) include the following:

  • Support for local domain indexes on hash- and interval-partitioned tables.

    These indexes are equipartitioned with the base table when base table is partitioned.

    See Also:

    • Partitioned Domain Indexes

    • Partitioning Domain Indexes

    • Input Data Partitioning

    • Restrictions of System-Managed Domain Indexing

    • Table 18-4

  • Support for an additional credential option and a directory object option to the CREATE LIBRARY syntax.
