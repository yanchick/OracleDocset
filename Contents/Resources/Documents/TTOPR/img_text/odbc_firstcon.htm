Description of the illustration odbc_firstcon.gif

The First Connection tab of the TimesTen ODBC Setup dialog box has two columns of fields and drop-down lists. The left column contains the following fields and drop-down lists: Permanent Data Size, Temporary Data Size, Replication Parallelism Buffer MB (with a default value of 64), Log Flush Method (drop-down list with 0-Platform Default chosen), Log Files Size (MB), Log Buffer Size (MB), Ckpt Frequency (secs), Ckpt Log Volume, and Ckpt Read Threads.

The right column contains the following fields and drop-down lists: Ckpt Rate (MB/sec), Recovery Threads, Connections, Log Buffer Parallelism, and Receiver Threads.

The following check-boxes appear below the fields in the right column: Auto Create, Force Connect, Log Auto Truncate, LogPurge, Overwrite, Preallocate. Auto Create, Log Auto Truncate, and Log Purge are checked.

The following buttons appear at the bottom of the dialog box: OK, Cancel, Default, Help.
