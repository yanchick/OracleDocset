Description of the illustration co.gif

The image co.gif shows the General tab page for collation (linguistic sort). The text fields and their contents are as follows:

  • Collation Name: MY_GENERIC_M

  • Collation ID: 10001

The Show Existing Definitions button is below the text fields. Below the button is a section called Defined Collation Flags. It contains the following radio buttons: CANONICAL_EQUIVALENCE, REVERSE_SECONDARY, SWAP_WITH_NEXT. CANONICAL_EQUIVALENCE is chosen.

There is a toolbar on the left side of the screen. The following menus are at the top of the screen: File, Edit, Tools, Help.
