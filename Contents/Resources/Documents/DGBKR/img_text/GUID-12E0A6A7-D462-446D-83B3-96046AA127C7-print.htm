Description of the illustration dgbkr001.eps

This figure shows an Oracle RAC database with three instances. Each instance defines the DG_BROKER_CONFIG_FILE1 and DG_BROKER_CONFIG_FILE2 initialization parameters that point to the ASM disk group. Each initialization parameter in the figure is pointing to its appropriate .dat file in the ASM disk group.
