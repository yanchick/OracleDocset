Description of the illustration accuserproperties.gif

This figure shows the Create Administrator: Properties page. To the right of the title there are the following buttons: Cancel, Next, and Review. Below the title there are the following fields: *Name, *Password, *Confirm Password, Password Profile, E-mail Address, Contact, Location, Department, Cost Center, Line of Business, and Description. Below these fields there is a Super Administrator check box. The Next button is selected.
