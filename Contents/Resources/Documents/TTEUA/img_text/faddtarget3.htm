Description of the illustration faddtarget3.gif

This figure shows the Add TimesTen Database page. To the right of the title there are the following buttons: Test Connection, OK, and Cancel. Below the title there are the following fields: Target Name, Target Type, Agent, Credential Type, * TimesTen database user name, * TimesTen password, * Confirm TimesTen password, Cache administration/cache manager user name, Oracle database cache administration password, Confirm Oracle database cache administration password, TimesTen database cache manager password, Confirm TimesTen database cache manager password, * Data Source Name, and *TimesTen instance name. The Target Type, Agent, and Credential type fields are auto-filled. In the top right corner there are the following buttons: Test Connection, OK, and Cancel. The Test Connection button is selected.
