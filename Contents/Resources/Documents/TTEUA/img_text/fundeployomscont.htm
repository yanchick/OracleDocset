Description of the illustration fundeployomscont.gif

This figure shows the Undeploy Plug-in From Management Server General dialog. Below the title there are the following fields: Name, Version, and Repository SYS Password. The Name and Version fields are auto-filled. Below the fields is a Target Types table with the following columns: Name and Supported Target Versions. At the bottom of the dialog are the Continue and Cancel buttons. The Continue button is selected.
