Description of the illustration fconfiguration.gif

This figure shows the configuration region of the TimesTen target. Below the title there are the following fields: Allocated PERM Space, Allocated TEMP Space, Internal Log Buffer Size, and PL/SQL Memory Segment Size. These fields are auto-filled. Each field also indicates the unit size of the corresponding field.
