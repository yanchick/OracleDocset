Description of the illustration fresthistorysel.gif

This figure shows the Restore History tab. There are three tabs at the top: Backup Schedule, Backup History and Restore Database, and Restore History. The Restore History tab is selected. Below the tabs there are the following buttons: View, Delete, and Detach. Below the buttons there is a table with the following columns: Backup ID, Initiated By, Restore Start Date, Restore End Date, Status, Reason for Failure, Backup Start Date, Backup Type, Backup Directory, Backup File Prefix, Recurrence Interval, Backup Size (GB), and Elapsed Time (secs).
