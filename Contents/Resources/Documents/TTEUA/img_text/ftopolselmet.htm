Description of the illustration ftopolselmet.gif

This figure shows the Choose Metrics dialog. Below the title there is a * Target Type drop-down list. Below the * Target Type drop-down list there is a View button and a Search field. Below the Search field there is a list of all of the available metrics for your selected target type. Below the metrics list there are the following buttons: Metric Order, Restore to Defaults, OK, and Cancel.
