Description of the illustration fchkpthist.gif

This figure shows the Checkpoint History region of your TimesTen database target. At the top of the region there are two buttons: View and Detach. Below these buttons there is a table with the following columns: Identifiers, Type, Initiator, Start Time, End Time, Dirty Blocks, Database file number, Checkpoint Volume (MB), Rate (MB/sec), and Status.
