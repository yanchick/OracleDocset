Description of the illustration fdeployagent4.gif

This figure shows the Deploy Plug-in on Management Agent Select Management Agents dialog. Below the title there are the following fields: Name and Version. The fields are auto-filled. Below the fields is a Management Agents table with the following columns: Management Agent, Agent Version, Operating System, Deployed Version, and Status. At the bottom of the dialog are the Back, Continue, and Cancel buttons. The Continue button is selected.
