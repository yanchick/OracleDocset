Description of the illustration fawttracks.gif

This figure shows the Parallel AWT Tracks of the Cache Parallel AWT tab of the TimesTen Cache Synchronization Metrics page. There is a table with the following columns: Peer Name, Transactions Processed, SQL Mode Batches, PLSQL Mode Batches, Oracle Database Rollbacks, Batches Sent Fully, Batches Sent Partially, Batches Begin waited, Depend Begin Waits Count, and Depend Commit Waits Count.
