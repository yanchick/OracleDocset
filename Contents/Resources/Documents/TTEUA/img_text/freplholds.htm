Description of the illustration freplholds.gif

This figure shows the Replication Log Holds region of the TimesTen Replication Monitor page. There are five tabs at the top: Replication Peers, Replication Log Holds, Transmitter Thread Status, Receiver Threads Status, and Tracks. The Replication Log Holds tab is selected. Below the tabs there is a table with the following columns: Peer Name, Hostname, Last Message Sent, Log File Number, Latency, and Track.
