Description of the illustration ftopolview.gif

This figure shows the topology page. At the top of the page there is the name of the TimesTen instance target. Below the title there is the TimesTen Instance Home menu. Below the TimesTen Instance Home menu there is a Configuration Topology region with the following menu items: View drop-down list, Customize button, Go to Topology for Selected Target button (this button is not available), Find Up Targets button, Find search field, Options drop-down list, and Toggle Display of Sidebar button. The View drop-down list is expanded and has the following options: Uses and Used by. The Used By option is selected.
