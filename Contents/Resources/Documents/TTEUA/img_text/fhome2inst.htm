Description of the illustration fhome2inst.gif

This figure shows Enterprise Manager Cloud Control menu choices as well as menu choices that have been customized for TimesTen instance targets. The TimesTen Instance Home menu is expanded and shows the following choices: Home, Monitoring, Control, Job Activity, Information Publisher Reports, Configuration, Compliance, Target Setup, Target Sitemap, and Target Information. The Control option is selected and shows the following choices: Start/Stop Services, Create Blackout, and End Blackout.
