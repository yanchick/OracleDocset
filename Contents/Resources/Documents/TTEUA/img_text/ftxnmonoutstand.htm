Description of the illustration ftxnmonoutstand.gif

This figure shows the Transaction Information region. There are two tabs at the top: Outstanding locks and Awaiting locks (0). The Outstanding locks tab is selected. Below the tabs there are the following buttons: View, Query By Example, and Detach. Below the buttons there is a table with the following columns: Status, Program, Process Number, Context, Connection Id, and Transaction Id.
