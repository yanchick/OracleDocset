Description of the illustration accselpriv.gif

This figure shows the Target Privileges region of the Create Role: Target Privileges page. Below the region title there is a search region with the following items: Name field, Type drop-down list, Go button, and Clear button. Below the search region there is an Add button. The Add button is selected. Below the Add button there is a table with the following columns: Select, Name, Type, Manage Target Privileges Grants, Is Aggregate Target, and Is Privilege Propagation Enabled. Below the table there are the following buttons: Cancel, Back, Next, and Review.
