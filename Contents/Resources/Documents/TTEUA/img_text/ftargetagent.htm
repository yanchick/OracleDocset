Description of the illustration ftargetagent.gif

This figure shows the Search and Select: Targets - Oracle Enterprise Manager Search and Select: Targets dialog. Below the title there are the following fields: Target Type, Target Name, On Host, and Configuration Search. The Target Type field is auto-filled. Below the fields is a table with the following columns: Target Name, Target Type, On Host, and Status. At the bottom of the dialog are the Select and Cancel buttons. The Select button is selected.
