Description of the illustration fdbcontrolloadram.gif

This figure shows the Load/Unload RAM region. The top area shows the Load Status and Database Loaded Since fields of the TimesTen database and there are the following buttons: Load and Unload. Below this there is a Current Connections region with the following buttons: View, Query By Example, and Detach. Below the buttons there is a Current Connections table with the following columns: Name, Type, Process ID, Connection ID, and Context.
