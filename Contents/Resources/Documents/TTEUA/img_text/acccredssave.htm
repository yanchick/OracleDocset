Description of the illustration acccredssave.gif

This figure shows the Security Manage Access page. Below the title there is an Access Control region. The Access Control region contains a Grantee search region with the following buttons: Search and Clear. Below the Grantee search region there are the following buttons: Add Grant, Change Privilege(s) (this button is not available), and Revoke Grant (this button is not available). Below the buttons there is a table with the following columns: Grantee, Grantee Type, and Privileges. In the top right corner of the page there are the following buttons: Save and Cancel. The Save button is selected.
