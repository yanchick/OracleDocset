Description of the illustration fmonconfig.gif

This figure shows a table with the following columns: Target Name, Target Type, and Target Status. The TimesTen target is right clicked and shows the following choices: Home, Monitoring, Control, Job Activity, Information Publisher Reports, Availability, Configuration, Compliance, Target Setup, Target Sitemap, and Target Information. The Target Setup option is selected and shows the following choices: Monitoring Configuration, Administrator Access, Remove Target, Add to Group, and Properties. The Monitoring Configuration option is selected.
