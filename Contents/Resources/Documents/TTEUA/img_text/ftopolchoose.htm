Description of the illustration ftopolchoose.gif

This figure shows the topology page. At the top of the page there are the following menu items: Customize button, Go to Topology for Selected Target button (this button is not available), Find Up Targets button, Find search field, Options drop-down list, and Toggle Display of Sidebar button. The Options drop-down list is expanded and shows the following options: Annotations, Expand All (this option is not available), Collapse All (this option is not available), Layout, Reload Topology, and Print. The annotations option is expanded and shows the following options: Status, Incidents (this option is not available), Link Labels, Metric Values, Metric Names and Values, No Metrics, and Choose Metrics. The Choose Metrics option is selected.
