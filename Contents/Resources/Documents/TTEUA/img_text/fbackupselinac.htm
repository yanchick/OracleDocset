Description of the illustration fbackupselinac.gif

This figure shows the Backup Schedule tab. There are three tabs at the top: Backup Schedule, Backup History and Restore Database, and Restore History. The Backup Schedule tab is selected. Below the tabs there are the following buttons: View, Create, Activate/Deactivate, Immediate, Delete, and Detach. Below the buttons there is a table with the following columns: Backup ID, Backup Type, Backup Directory, Backup File Prefix, Recurrence Interval, Start Date, End Date, Schedule Start, Current Count, Maximum, Current Count, Maximum, Owner, Current State, and Last Successful Backup.
