Description of the illustration configdischostrun.gif

This figure shows the Setup Discovery page. Below the title there is an Instruction region with the following tabs: Host and Oracle VM Manager, Targets on Hosts, and Advanced: Discovery Modules. Below the tabs there is a Search region that contains the following buttons: View, Collection Schedule, Diagnostic Details, Discover Now, Discovery Modules, and Detach. Below the buttons there is a table with the following columns: Agent Host Name, Collection Schedule, Discovered Targets, Managed Targets, and Enabled Discovery Modules. The Discover Now button is selected.
