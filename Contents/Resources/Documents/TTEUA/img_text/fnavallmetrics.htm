Description of the illustration fnavallmetrics.gif

This figure shows Enterprise Manager Cloud Control menu choices as well as menu choices that have been customized for TimesTen database targets. The TimesTen Database Home menu is expanded and shows the following choices: Home, Monitoring, Control, Job Activity, Information Publisher Reports, Availability, Configuration, Compliance, Target Setup, Target Sitemap, and Target Information. The Monitoring option is selected and shows the following choices: Performance Summary, Transaction Monitor, Client/Server Summary, Cache Synchronization Metrics, Replication Monitor, All Metrics, Metric and Collection Settings, Metric Collection Errors, Status History, Incident Manager, Alert History, and Blackouts. The All Metrics option is selected.
