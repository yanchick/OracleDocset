Description of the illustration fbackupdeleteok.gif

This figure shows the Delete Backup Schedule dialog. There are the following fields: Backup ID, Backup Type, Backup Directory, Backup File Prefix, Recurrence Interval, Start Date, End Date, Schedule State, and Owner. All of the fields are auto-filled. At the bottom of the dialog there are the following buttons: OK and Cancel. The OK button is selected.
