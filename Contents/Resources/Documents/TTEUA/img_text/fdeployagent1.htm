Description of the illustration fdeployagent1.gif

This figure shows the information for opening the Enterprise Manager Plug-ins page. The Setup menu is expanded and shows the following choices: Add Target, Extensibility, Proxy Settings, Security, Incidents, Notifications, Provisioning and Patching, My Oracle Support, Middleware Management, Manage Cloud Control, Command Line Interface, and Management Packs. The Extensibility option is selected and shows the following choices: Self Update, Plug-ins, Management Connectors, Data Exchange, and Development Kit. The Plug-ins option is selected.
