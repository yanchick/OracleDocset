Description of the illustration ftxnmonawait.gif

This figure shows the Awaiting locks region. There are two tabs at the top: Outstanding locks and Awaiting locks (2). The Awaiting locks (2) tab is selected. Below the tabs there are the following buttons: View, Query By Example, and Detach. Below the buttons there is a table with the following columns: Resource, Resource Id, On Table, Process Number, Context, Transaction Id, Mode, SQL Text, Holder Transaction Id, Holder Mode, and Holder SQL Text.
