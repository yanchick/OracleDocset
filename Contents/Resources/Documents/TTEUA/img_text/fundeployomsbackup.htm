Description of the illustration fundeployomsbackup.gif

This figure shows the Undeploy Plug-in From Management Server dialog. Below the title there is a Have you backed up the repository and configuration of first management server? check box. Below the check box there is the following fields: Name and Version. These fields are auto-filled. Below these fields is a Management Servers table with the following columns: Name, Currently Deployed Plug-in Version and Status. At the bottom of the dialog are the Back, Undeploy, and Cancel buttons. The Have you backed up the repository and configuration of first management server? check box is selected.
