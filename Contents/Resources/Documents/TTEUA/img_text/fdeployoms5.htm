Description of the illustration fdeployoms5.gif

This figure shows the Deploy Plug-ins on Management Servers: Review page. Below the title there is the Management Servers table with the following columns: Name and Status. Below the Management Servers table is the Plug-ins table with the following columns: Name, Latest Available, Currently Deployed, To Deploy, and Downtime Required. At the top right of the page are the following buttons: Back, Next, and Cancel. The Deploy button is selected.
