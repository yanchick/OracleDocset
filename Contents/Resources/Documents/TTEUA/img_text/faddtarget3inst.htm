Description of the illustration faddtarget3inst.gif

This figure shows the Add: TimesTen Instance page. To the right of the title there are the following buttons: Test Connection, OK, and Cancel. Below the title there are the following fields: Target Name, Target Type, Agent, and TimesTen instance name. The Target Type and Agent fields are auto-filled. The Test Connection button is selected.
