Description of the illustration fcacheauto.gif

This figure shows the Cache Autorefresh region of the TimesTen Cache Synchronization Metrics page. Below the title there are the following fields: Cycles completed successfully on TimesTen, Number of cycles that failed due to errors, Rows deleted in TimesTen from Oracle Database, Rows inserted in TimesTen from Oracle Database, Rows updated in TimesTen from Oracle Database, Full refreshes triggered, and Database Loaded Since. These fields are auto-filled.
