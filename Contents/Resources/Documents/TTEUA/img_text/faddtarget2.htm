Description of the illustration faddtarget2.gif

This figure shows the Add Targets Manually page. Below the Add Targets Manually title there are the following options: Add Host Targets, Add Targets Using Guided Process, and Add Targets Declaratively by Specifying Targets Monitoring Properties. The Add Targets Declaratively by Specifying Targets Monitoring Properties option is selected. Below these options there is a Target Type drop-down list and TimesTen Database is selected. Below the Target Type drop-down list is a Monitoring Agent field. At the bottom of the dialog there is an Add Manually button (this button is not available).
