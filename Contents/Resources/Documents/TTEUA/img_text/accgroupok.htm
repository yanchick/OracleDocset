Description of the illustration accgroupok.gif

This figure shows the Add Target Create Privilege Propagating Group page. Below the title there are four tabs: General, Charts, Columns, Dashboard, and Access. The General tab is selected. In the General tab there are the following items: *Name text box, Privilege Propagation check box, and Parent Groups text field. Below the Parent Groups field there is a Members region with the following buttons: Remove, Add, and Add By Criteria. Below the buttons there is a Members table. In the top right corner of the page there are the following buttons: OK and Cancel. The OK button is selected.
