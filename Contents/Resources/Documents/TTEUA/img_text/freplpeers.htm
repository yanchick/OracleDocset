Description of the illustration freplpeers.gif

This figure shows the Replication peers region of the TimesTen Replication Monitor page. There are five tabs at the top: Replication Peers, Replication Log Holds, Transmitter Thread Status, Receiver Thread Status, and Tracks. The Replication Peers tab is selected. Below the tabs there are the following buttons: View and Detach. Below the button there is a table with the following columns: Name, Hostname, State, and Port and Status.
