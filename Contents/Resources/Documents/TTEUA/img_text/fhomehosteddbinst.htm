Description of the illustration fhomehosteddbinst.gif

This figure shows the Hosted Databases region of your TimesTen instance target page. Below the title there are the following buttons: View, Details, and Detach. Below the button there is a table with the following columns: Database, Total Connection, Client Server Connections, RAM Status, Cache Agent, and Replication Agent.
