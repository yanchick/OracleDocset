Description of the illustration freplreceiver.gif

This figure shows the Replication Receivers region of the Receiver Thread Status tab of the TimesTen Replication Monitor page. There are five tabs at the top: Replication Peers, Replication Log Holds, Transmitter Thread Status, Receiver Thread Status, and Tracks. The Receiver Threads Status tab is selected. Below the tabs there is a table with the following columns: Peer name, Start or Restart count, and Transactions Received.
