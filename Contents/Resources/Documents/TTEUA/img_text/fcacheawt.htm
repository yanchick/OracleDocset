Description of the illustration fcacheawt.gif

This figure shows the Cache AWT region of the TimesTen Cache Synchronization Metrics page. There are three panes in the Cache AWT region: General, SQL Array Execution Method, and PL/SQL Execution Method.

In the General pane there are the following fields: Transactions committed on Oracle database, Calls made to Oracle database, Rollbacks on Oracle database because of errors, Transactions propagated to Oracle database, and Transactions retried in case of an error. In the SQL Array Execution Method pane there are the following fields: Batches sent to Oracle database, Bytes sent to Oracle database, Delete batches sent to Oracle database, Rows deleted on Oracle database, Insert batches sent to Oracle database, Rows inserted on Oracle database, Update batches sent to Oracle database, and Rows updated on Oracle database. In the PL/SQL Execution Method pane there are the following fields: Batches sent to Oracle database, Bytes sent to Oracle database, Rows deleted on Oracle database, Rows inserted on Oracle database, Rows updated on Oracle database, and Database Loaded Since. All of these fields are auto-filled.
