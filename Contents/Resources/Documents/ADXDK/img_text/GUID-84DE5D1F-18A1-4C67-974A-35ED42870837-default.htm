Description of the illustration c7.gif

Library Directories window appears. It has a field for entering the new library directory. Above this is the tool bar which contains menu icons to Add New Line, Cancel, and up or down arrows for moving from one existing Library Directory to another. At the bottom of the window are OK and Cancel buttons and a Macros button, leading to lists of Macros.
