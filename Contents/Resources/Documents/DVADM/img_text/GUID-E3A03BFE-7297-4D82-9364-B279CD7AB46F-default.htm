Description of the illustration reports1.gif

Image reports.gif shows the opening page of the Reports page. On the left, from top to bottom, are the following components:

  • Database Vault Configuration Issues, with the following reports under it: Command Rule Configuration Issues, Rule Set Configuration Issues, Realm Authorization Configuration Issues, Factor Configuration Issues, and Identity Configuration Issues

  • Database Vault Enforcement Audit Reports

  • Database Vault Configuration Changes

To the right of this pane is a pane for the currently selected report. From top to bottom are the following components:

  • The name of the selected report. At the far right is the OK button.

  • A brief description of the purpose of the report

  • A Search area with settings appropriate for the current report. At the far right are the Search and Reset buttons.

  • A table containing the report's contents, which are as follows:

    • A View menu, which contains the items Columns (with a submenu with Show All, feature_name Name, and Issue), Detach, and Reorder Columns; Export to Spreadsheet, and Detach

    • The columns in the table are named depending on the needs of the feature.
