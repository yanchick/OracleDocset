Description of the illustration priv_analysis_report.gif

Image priv_analysis_reports.gif shows the Reports page. From left to right, top to bottom, are the following components:

  • Page Summary, Unused, and Used tabs.

  • A Search field, which has been expanded to show a list to select the policy and a second list to show the grantees affected by the policy.

  • A Revoke button and a Re-grant button.

  • A table showing the contents of the report. From left to right are the columns Grantee, Type (that is, User, Folder, or Role), and Used. To the right of these columns are Used and Unused columns for System Privileges and for Object Privileges. To the right of these is the Revoke column, which indicates if the role or privilege has been revoked.
