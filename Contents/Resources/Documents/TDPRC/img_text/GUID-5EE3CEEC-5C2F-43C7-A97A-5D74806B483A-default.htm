Description of the illustration cc12_clstr_intercon_cln.png

The screenshot shows the Cluster Interconnects page.

At the beginning of the page is the following introductory text: "The interconnect configuration and internode communication will influence the performance of cluster databases. The tables below show network interfaces on all hosts and network interfaces currently in use by cluster databases. It is important that cluster databases are configured to use a private interconnect for message and block transfers." Below this text is the Private Interconnect Transfer Rate (MB/Sec), which has a current value of 0.091. This is described as "Transfer rate on the private network in the last 5 minutes."

There are 2 tables shown in the screenshot. They are the Interfaces by Hosts table and the Interfaces in Use by Cluster Databases table.

At the top of the Interfaces by Hosts table is the View action list, which has the value Private selected. Below this action list are two links: Expand All and Collapse All. The table has 6 columns and 3 rows. The column names, from left to right, are: Name, Type, Subnet, Interface Type, Total I/O Rate (MB/Sec) (Last 5 Minutes), and Total Error Rate (%) (Last 5 Minutes). The values contained in the rows are, from top to bottom:

  • racbiclstr (preceded by a downward pointing triangle, indicating the entry has been expanded), Cluster, and no further values

  • racbi05.example.com (hyperlinked, and preceded by a downward pointing triangle, indicating the entry has been expanded, Host, and no further values

  • eth2 (hyperlinked), Interface, 10.0.0.8, Private, .191 (hyperlinked), and 0 (hyperlinked)

The Interfaces in Use By Cluster Databases table has two hyperlinks at the top: Expand All and Collapse All. The table has 8 columns and 6 rows. The column names, from left to right, are: Name, Target Type, Interface Name, Host Name, IP Address, Interface Type, Source, and Transfer Rate (MB/Sec) (Last 5 Minutes). The rows contain the following values:

  • racbiclstr (preceded by a downward pointing triangle, indicating the entry has been expanded), Cluster, and no further values

  • qos1 (hyperlinked and preceded by a downward pointing triangle, indicating the entry has been expanded), Cluster Database, and no further values

  • qos1_1 (hyperlinked), Database Instance, eth2:1 (hyperlinked), racbi06.example.com (hyperlinked), 192.0.2.234, Private (hyperlinked), n/a, .007 (hyperlinked)

  • qos1_3 (hyperlinked), Database Instance, eth2:1 (hyperlinked), racbi05.example.com (hyperlinked), 198.51.100.24, Private (hyperlinked), n/a, n/a

  • qos2 (hyperlinked and preceded by a triangle pointing to the right, indicating the entry has been collapsed), Cluster Database, and no further values

  • qos3 (hyperlinked and preceded by a triangle pointing to the right, indicating the entry has been collapsed), Cluster Database, and no further values

Below this table is the following text:

TIP The Transfer Rate is the estimated traffic contributed by the instance assuming uniform block size in the database.

End of description.
