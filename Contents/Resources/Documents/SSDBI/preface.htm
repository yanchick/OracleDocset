Preface

This guide explains how to install and configure Oracle Database for Oracle Solaris on SPARC (64-Bit) and for Oracle Solaris on x86-64 (64-Bit). This guide also provides information about Optimal Flexible Architecture, cloning an Oracle home, troubleshooting the installation, and how to remove the database software..

The preface contains the following topics:

  • Audience

  • Documentation Accessibility

  • Related Documentation

  • Conventions

Audience

This guide is intended for anyone responsible for installing Oracle Database 12c Release 1 (12.1) on Oracle Solaris systems. Additional installation guides for Oracle Database, Oracle Real Application Clusters, Oracle Clusterware, Oracle Database Examples, and Oracle Enterprise Manager Cloud Control are available on Oracle Technology Network at:
