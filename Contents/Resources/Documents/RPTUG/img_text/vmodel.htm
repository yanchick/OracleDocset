Description of the illustration vmodel.eps

This is a text description of Figure 2-2, "V-model with a Database Migration". It illustrates the process of using V-model with a database migration scenario, as explained in text before the illustration. The two boxes at the top level are Development Phase and Testing Phase. The two boxes at the next level down are Database Migration Requirements and System Acceptance Testing. The two boxes at the next level down are Database/Application Design and Integration Testing. The box at the bottom level is Database Object Testing.
