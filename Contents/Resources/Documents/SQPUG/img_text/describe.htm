Description of the illustration describe.gif

The describe.gif image shows a HTML table of three columns, generated from iSQL*Plus. It contains the output from the DESCRIBE command in the text prior to this image. The heading row has column headings of Name, Null? and Type. There are 16 rows of output for the EMP_DETAILS_VIEW table view.
