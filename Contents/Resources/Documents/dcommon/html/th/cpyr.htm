ข้อสงวนสิทธิ์เกี่ยวกับการรับประกันข้อจำกัดของใบอนุญาต/ความเสียหายอันเป็นผลสืบเนื่อง

ซอฟต์แวร์นี้และเอกสารประกอบที่เกี่ยวข้องมีให้ตามข้อตกลงของใบอนุญาตที่ประกอบด้วยข้อจำกัดในการใช้และการเปิดเผย และได้รับความคุ้มครองตามกฎหมายทรัพย์สินทางปัญญา ยกเว้นกรณีที่อนุญาตไว้อย่างชัดเจนในข้อตกลงของใบอนุญาตของคุณ หรือที่กฎหมายกำหนด ห้ามมิให้ใช้ คัดลอก ทำซ้ำ แปล ประชาสัมพันธ์ ดัดแปลง ให้การอนุญาต ส่ง แจกจ่าย แสดง ดำเนินการ เผยแพร่ หรือจัดแสดงบางส่วน ไม่ว่าในรูปแบบและวิธีการใดๆ ห้ามมิให้ย้อนกระบวนการผลิต ย้อนการประกอบ หรือย้อนกระบวนการคอมไพล์ซอฟต์แวร์นี้ ยกเว้นกรณีที่กฎหมายกำหนดให้ทำได้เพื่อให้สามารถทำงานร่วมกัน

ข้อสงวนสิทธิ์เกี่ยวกับการรับประกัน

ข้อมูลที่มีอยู่ในที่นี้อาจมีการเปลี่ยนแปลงโดยไม่ต้องแจ้งให้ทราบและไม่มีการรับประกันว่าปราศจากข้อผิดพลาด ถ้าคุณพบข้อผิดพลาดใด โปรดรายงานให้เราทราบอย่างเป็นลายลักษณ์อักษร

คำประกาศเกี่ยวกับการใช้งานที่เป็นอันตราย

ซอฟต์แวร์หรือฮาร์ดแวร์นี้ได้รับการพัฒนาขึ้นเพื่อการใช้งานทั่วไป เพื่อวัตถุประสงค์ในการจัดการข้อมูลในรูปแบบต่างๆ ไม่ได้มีขึ้นเพื่อใช้ในงานที่มีอันตราย รวมถึงงานที่อาจทำให้เกิดความเสี่ยงที่จะบาดเจ็บ ถ้าคุณใช้ซอฟต์แวร์หรือฮาร์ดแวร์นี้ในการทำงานที่เป็นอันตราย คุณจะต้องรับผิดชอบต่อการใช้มาตรการป้องกัน สำรองข้อมูล ระบบสำรอง และมาตรการอื่นๆ เพื่อให้มั่นใจได้ถึงความปลอดภัยในการใช้งาน Oracle Corporation และบริษัทในเครือไม่มีความรับผิดต่อความเสียหายที่เกิดจากการใช้ซอฟต์แวร์หรือฮาร์ดแวร์นี้ในงานที่เป็นอันตราย

คำประกาศเกี่ยวกับเครื่องหมายการค้า

Oracle และ Java เป็นเครื่องหมายการค้าจดทะเบียนของ Oracle และ/หรือบริษัทในเครือ ชื่ออื่นอาจเป็นเครื่องหมายการค้าของผู้ที่เป็นเจ้าของ

Intel และ Intel Xeon เป็นเครื่องหมายการค้าหรือเครื่องหมายการค้าจดทะเบียนของ Intel Corporation เครื่องหมายการค้าของ SPARC ทั้งหมดใช้โดยได้รับการอนุญาตและเป็นเครื่องหมายการค้าหรือเครื่องหมายการค้าจดทะเบียนของ SPARC International, Inc. AMD, Opteron, โลโก้ AMD และโลโก้ AMD Opteron เป็นเครื่องหมายการค้าหรือเครื่องหมายการค้าจดทะเบียนของ Advanced Micro Devices UNIX เป็นเครื่องหมายการค้าจดทะเบียนของ The Open Group

ข้อสงวนสิทธิ์เกี่ยวกับเนื้อหา ผลิตภัณฑ์ หรือบริการของบุคคลที่สาม

ซอฟต์แวร์หรือฮาร์ดแวร์นี้และเอกสารประกอบอาจมีการเข้าใช้หรือข้อมูลเกี่ยวกับเนื้อหา ผลิตภัณฑ์ และบริการจากบุคคลที่สาม Oracle Corporation และบริษัทในเครือไม่มีส่วนรับผิดชอบ และปฏิเสธการรับประกันโดยชัดแจ้งในทุกกรณี เกี่ยวกับเนื้อหา ผลิตภัณฑ์ และบริการของบุคคลที่สาม เว้นแต่จะมีการกำหนดไว้ในข้อตกลงที่ใช้บังคับระหว่างคุณกับ Oracle ทั้งนี้ Oracle Corporation และบริษัทในเครือไม่มีส่วนรับผิดชอบต่อการสูญเสีย ต้นทุน หรือความเสียหายที่เกิดขึ้นเนื่องจากการเข้าใช้หรือใช้งานเนื้อหา ผลิตภัณฑ์ หรือบริการของบุคคลที่สาม เว้นแต่จะมีการระบุไว้ในข้อตกลงที่ใช้บังคับระหว่างคุณกับ Oracle

คำประกาศเกี่ยวกับเอกสารร่างก่อนใช้งานจริง

หากเอกสารนี้มีสถานะก่อนใช้งานจริง:

เอกสารนี้มีสถานะก่อนใช้งานจริง และมีให้เพื่อการสาธิตและการใช้งานในเบื้องต้นเท่านั้น อาจไม่ใช่ข้อมูลที่เจาะจงสำหรับฮาร์ดแวร์ที่คุณใช้ร่วมกับซอฟต์แวร์นี้ Oracle Corporation และบริษัทในเครือไม่มีส่วนรับผิดชอบ และปฏิเสธการรับประกันโดยชัดแจ้งในทุกกรณีเกี่ยวกับเอกสารนี้ และจะไม่มีส่วนรับผิดชอบต่อการสูญเสีย ต้นทุน หรือความเสียหายที่เกิดขึ้นจากการใช้เอกสารนี้

คำประกาศเกี่ยวกับเอกสารร่างก่อนใช้งานจริงส่วนบุคคล

หากเอกสารนี้มีสถานะก่อนใช้งานจริงส่วนบุคคล:

ข้อมูลที่มีอยู่ในเอกสารนี้มีวัตถุประสงค์เพื่อการแบ่งปันทางข้อมูลเท่านั้นและควรได้รับการพิจารณาตามความสามารถของคุณในฐานะที่เป็นสมาชิกของคณะกรรมการให้คำปรึกษาของลูกค้า หรือตามข้อตกลงการทดลองรุ่นเบต้าของคุณเท่านั้น ทั้งนี้ ไม่แสดงถึงข้อผูกมัดที่จะจัดให้มีเนื้อหา รหัส หรือฟังก์ชันการทำงาน และไม่ควรนำมาใช้อ้างอิงเพื่อตัดสินใจเกี่ยวกับการจัดซื้อ การพัฒนา การรีลีส และระยะเวลาของคุณสมบัติหรือฟังก์ชันที่อธิบายในเอกสารนี้ ขึ้นอยู่กับการพิจารณาของ Oracle แต่เพียงผู้เดียว

เอกสารนี้ไม่ว่าจะอยู่ในรูปแบบใดๆ ทั้งที่เป็นซอฟต์แวร์ หรือเอกสารสิ่งพิมพ์ใดๆ มีข้อมูลที่เป็นเอกสิทธิ์ซึ่งเป็นทรัพย์สินเฉพาะของ Oracle การเข้าใช้และใช้งานเอกสารลับเฉพาะนี้ต้องเป็นไปตามข้อกำหนดและเงื่อนไขของข้อตกลงหลักของ Oracle, ข้อตกลงการอนุญาตใช้งานและการให้บริการของ Oracle, ข้อตกลง PartnerNetwork ของ Oracle, ข้อตกลงการกระจายของ Oracle หรือข้อตกลงการอนุญาตใช้งานอื่นที่มีการลงนามระหว่างคุณกับ Oracle ซึ่งคุณยอมรับที่จะปฏิบัติตาม ห้ามเปิดเผย คัดลอก ทำซ้ำ หรือแจกจ่ายเอกสารนี้และข้อมูลในเอกสารนี้แก่บุคคลภายนอก Oracle หากไม่ได้รับอนุญาตอย่างเป็นลายลักษณ์อักษรจาก Oracle เอกสารนี้ไม่ใช่ส่วนหนึ่งของข้อตกลงการอนุญาต และไม่สามารถรวมไว้ในข้อตกลงที่เป็นสัญญาใดๆ กับ Oracle หรือบริษัทย่อยหรือบริษัทในเครือ

การเข้าใช้บริการด้านเทคนิคของ Oracle

ลูกค้า Oracle ที่ซื้อบริการสนับสนุนสามารถเข้าใช้การสนับสนุนแบบอิเล็กทรอนิกส์ผ่านทาง My Oracle Support โปรดดูข้อมูลได้ที่ http://www.oracle.com/pls/topic/lookup?ctx=acc&id=info หรือไปที่ http://www.oracle.com/pls/topic/lookup?ctx=acc&id=trs หากคุณมีปัญหาทางการได้ยิน
