
In the cpyr.htm file and the cpyr-css.htm file, the copyright notice contains a date range that starts with 1994 and ends with the current date, as in 1994-2016. Do not change the starting year, 1994. Even if your software and documentation were developed after 1994, do not change the starting year, 1994.

The cpyr.htm file and the cpyr-css.htm file differ in that the cpyr-css.htm file contains the following, which is not in the cpyr.htm file: 
<link rel="stylesheet" href="../css/blafdoc.css" type="text/css" />
Use the  cpyr-css.htm file if you want to link to a different stylesheet. Replace blafdoc.css with the name of your stylesheet.
