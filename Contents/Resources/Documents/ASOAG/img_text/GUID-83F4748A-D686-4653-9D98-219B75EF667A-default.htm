Description for em_enable_disable

Image em_red_enable_disable.png shows the Policies region of the Data Redaction page. From top to bottom, left to right, are the following components:

  • The label Policies

  • The buttons Create, Edit, View, Enable, Disable, Delete. Enable is not available because the currently selected policy is already enabled.

  • A table listing the policies, with the following fields, from left to right: Schema, Table/View, Policy Name, Enabled, Redacted Columns
