Description of em_create_red_pol.png

Image em_create_red_pol.png shows the Create Data Redaction Policy window with a policy created. From left to right, top to bottom, are the following components:

  • The breadcrumb database, CDB1_PDB1

  • The menu breadcrumb Oracle Database, Performance, Availability, Security, Schema, Administration

  • The label Create Data Redaction Policy: emp_wages_pol

  • The fields, top to bottom: Schema, Table/View, Policy Name, Policy Expression

  • To the right of the fields is an instruction box that describes the instructions in this section.

  • To the right of the instruction box are the buttons Cancel, Show SQL, OK.

  • Following the entry fields is a table entitled Object Columns, with the buttons (left to right) Add, Modify, Remove, followed by the table, which has the fields Column, Column Datatype, Redaction Function, Function Attributes.
