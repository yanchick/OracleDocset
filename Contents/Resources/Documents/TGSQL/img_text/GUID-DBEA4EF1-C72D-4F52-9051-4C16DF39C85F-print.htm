Description of the illustration tgsql_vm_049.eps

This graphic depicts two cylinders: Production Database, and Test Database. Production Database contains a box with the following contents: System-Supplied Schema, PACK_STGTAB_SQLSET. An arrow points from this box to another box that contains the text "Staging Table" and two intersecting lines. The Staging Table box points outside of the cylinder to a binary file labeled ".dump file." The arrow is labeled "Data Pump Export." An arrow points from the .dump file to an identical .dump file below the Test Database cylinder. The arrow is labeled "Transport ftp, nfs." The Test Database is identical to the production Database with the following exceptions: the arrows point in the opposite direction; the procedure is named UNPACK_STGTAB_SQLSET.
