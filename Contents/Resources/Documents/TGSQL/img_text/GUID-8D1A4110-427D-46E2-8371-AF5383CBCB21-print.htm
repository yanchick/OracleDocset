Description of the illustration tgsql_vm_039.eps

A box with the text "GATHER_TABLE_STATS" points down to an Optimizer icon. This icon points right to a box labeled "Optimizer Statistics." This box contains two boxes. One box has the callout "Published Statistics." The other box has the callout "Pending Statistics." A box labeled "Customers Table" points up to an Optimizer icon. The arrow is labeled "Publishing preferences set to FALSE." Underneath this set of icons is another set of icons. The Optimizer icon is pointed to by a box containing the text "SELECT ... FROM customers." A box containing the text "OPTIMIZER_USE_PENDING_STATISTICS=FALSE" points up to the Optimizer icon. The same Data Dictionary box that appears in the top set of icons is joined to the Optimizer icon by a double-headed arrow.
