Description of the illustration cachegrid.eps

The image shows a cache grid with two members that are standalone databases and one member that is an active standby pair. The active database sends updates to the standby database. The standby database sends updates to a read-onlhy subscriber database that is not part of the cache grid. The cache grid members send and receive updates from an Oracle database.

End of description.
