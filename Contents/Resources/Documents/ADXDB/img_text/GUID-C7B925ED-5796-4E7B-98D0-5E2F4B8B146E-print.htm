Description of the illustration admin075.eps

This figure shows a tree structure of folders, organized into five levels. At the root (top) of the tree is a folder labeled /sys/asm. It has two subfolders, labeled DATA and RECOVERY, on a level labeled Disk Groups. Each of these folders has two subfolders, labeled HR and MFG, on a level labeled Databases. The two MFG folders have the following subfolders, on a level labeled File Types: The leftmost MFG folder has subfolders labeled DATAFILE, TEMPFILE, CONTROLFILE, and ONLINELOG. The right MFG folder has subfolders labeled CONTROLFILE, ONLINELOG, and ARCHIVELOG. At the lowest level (unlabeled), each of the folders on level File Types has one or more files.
