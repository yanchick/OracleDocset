Description of the illustration sql_access_schedule.gif

The screenshot shows the SQL Access Advisor: Schedule page.

Under the page title are the following label-value pairs: Database is empdb, Logged In As is DBA1.

To the right of the label-value pairs are the Cancel, Back, and Next buttons. Between the Back and Next buttons is the text "Step 3 of 4".

The Advisor Task Information group has some fields and lists that require values and some that do not. The fields and lists are the following:

  • Task Name (required)

    The value in the field is SQLACCESS8895797.

  • Task Description

    The value in the field is SQL Access Advisor.

  • Journaling Level

    The value in the list is Basic. Text associated with the list is: "The level of journaling controls the amount of information that is logged to the advisor journal during execution of the task. This information appears on the Details tab when viewing task results."

  • Task Expiration (days) (required)

    The value in the field is 30. Text associated with the field is: "Number of days this task will be retained in the database before being purged."

  • Total Time Limit (minutes) (required)

    The value in the field is 10000.

The Scheduling Options group has the Schedule Type list, which has the value Standard selected. For the Standard type, the group contains the following groups, lists, and fields:

  • Time Zone

    This list has the value (UTC +00:00) Universal Time selected. To the right of the Time Zone list is a search icon.

  • Repeating

    This group has the Repeat list, which has the value Do Not Repeat selected.

  • Start

    This group has the following options.

    • Immediately (selected)

    • Later

      This option has the Date field and Time lists. The Date value is Feb 22, 2012. This field has an associated Calendar icon. Below the Date field are Time lists for hours, minutes, and seconds. The lists values show the time 8:30:00. To the right of these Time lists are the options AM and PM (selected).

End of description.
