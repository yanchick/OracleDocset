Description of the illustration baseline_snapshot_range.gif

The Baseline Interval section is shown.

The Snapshot Range option is selected.

The Change Chart Time Period navigation tree is not expanded.

In the Select Time Period section, the Period Start Time option is not selected. The value in the Period Start Time field is Feb 7, 2012 10:50:03 AM CST. The Period End Time option is selected. The value in the Period End Time field is Feb 7, 2012 11:20:013 AM CST.

A line chart is shown. The y-axis is labeled Active Session and ranges from 0 to 3.410. The x-axis shows times of day, shown in 2-hour intervals, ranging from 1:00 p.m. Feb 6, 2012 to 12:00 p.m. Feb 7, 2012. The chart tracks Wait, User I/O, and CPU. Beginning at 1:00 p.m. Feb 6, the activity rises from 0 to 1.137 for User I/O and CPU and to 2.274 for Wait at 2:00 p.m. Feb 6. The activity falls back to 0 by 2:30 p.m. Feb 6. The activity rises again from 0 at 7:45 p.m. Feb 7 and to about 1.600 for User I/O and CPU and to about 4.400 for Wait at 8:00 p.m. The activity then drops to 1.137 for all classes by 9:45 p.m. and remains at that level until 12:p.m. when all activity begins to go down.
