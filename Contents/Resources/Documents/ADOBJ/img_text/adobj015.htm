Description of the illustration adobj015.eps

The figure shows the people_objtab object table with lines connecting the table's attributes phones_ntab, address_obj, and name_obj to outlying schematics. These schematics show expanded listings of their attributes and types.

The figure is a graphical representation of the CREATE TYPE person_objtyp and CREATE TABLE people_objtab statements that precede the figure in the text.
