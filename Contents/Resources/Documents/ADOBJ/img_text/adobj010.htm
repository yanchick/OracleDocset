Description of the illustration adobj010.eps

The figure shows representations of the LineItemList_ntab nested table and the Stock_objtab object table. A line representing a reference connects the attribute Stock_ref of the LineItemList_ntab to the Stock_objtab table.

The figure is a graphical representation of the ALTER TABLE PoLine_ntab statement that precedes the figure in the text.
