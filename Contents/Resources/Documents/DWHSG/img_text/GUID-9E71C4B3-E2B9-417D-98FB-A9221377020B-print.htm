Description of the illustration dwhsg151.eps

This image contains a rectangle labeled Dimension Table CUSTOMERS on the left. A grey area within this rectangle indicates that this table contains an updated row.

In the center, placed one below the other, are two rectangles representing the partitions P1 and P2 of the Fact Table SALES. The partition P1 is divided into three parts labeled Z1, Z2, and Z3. The partition P2 is divided into two parts labeled Z4 and Z5. There is a line connecting CUSTOMERS to P1 and another line connecting CUSTOMERS to P2.

On the right is a rectangle representing the zone map table SALES_ZMAP. This rectangle is divided into seven portions that are labeled in the following order: Z2, Z3, Z5, Z4, Z1, P1, and P2. All the portions are marked in grey to indicate that they contain stale rows.
