Description of the illustration shortest_a_to_d.jpg

This illustration is an image of the output from the analytical functions used in Example 7-6 in Analytical Functions for RDF Data. The image shows four nodes, each representing a person, and lines with arrowheads showing the relationships between the persons (expressed as the predicates differs, likes, dislikes, and knows).
