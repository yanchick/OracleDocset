Description of the illustration sybase_migration.eps

Graphical representation of a Sybase Application running on Oracle Database

  • Sybase Application interfaces with Oracle Database's Sybase SQL Translation Profile through the Oracle ODBC/JDBC Driver.

  • Oracle Database's Sybase SQL Translation Profile uses the following artifacts stored and registered within Oracle Database:

    • Custom SQL Translations

    • Auto Translator

    • Custom Error-Code Mappings

  • Oracle Database's Sybase SQL Translation Profile reads/writes from/to Application Tables and Stored Procedures within Oracle Database.
