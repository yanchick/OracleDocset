Description of the illustration ndmdemo_sf.gif

This is a text description of Figure 5-1. It illustrates nodes and links from San Francisco network data, displayed in a network analysis demo application window.

The left side of the window includes areas for viewing and entering information related to shortest path analysis, network constraints, prohibited zone, link cost calculators, keep previous results, reverse direction, include traffic patterns, and start time, plus a Find Shortest Path button and a display of the analysis result. The middle and right area of the window contains a map representation of nodes and links.
