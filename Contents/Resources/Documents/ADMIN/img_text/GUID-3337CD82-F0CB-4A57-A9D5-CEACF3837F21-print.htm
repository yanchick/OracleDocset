Description of the illustration admin048.eps

The image admin048.gif shows a failure during the commit phase of a distributed transaction. User SCOTT, and two databases, Local and Remote, are shown. The Local database has COMMIT_POINT_STRENGTH = 200. The Remote database has COMMIT_POINT_STRENGTH = 100. The events leading to failure are described in the text following the image.
