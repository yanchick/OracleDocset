Description of the illustration rpt08repl.gif

This graphic shows replication statistics. For each subscriber (Repl_Peer), it shows begin and end values for Repl_Log_Send_LSN, Repl_Log_Behind, and Repl_Latency. It shows a number of values and rates of change for Repl_RPS and Repl_TPS. It also shows a number of Repl_Peer values.
