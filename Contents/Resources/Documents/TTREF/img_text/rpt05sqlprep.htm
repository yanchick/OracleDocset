Description of the illustration rpt05sqlprep.gif

This graphic shows the SQL Sort by Preparations section of a ttStats report. For the top 30 SQL commands that the application has used during the report period, it shows the number of preparations, the percentage that number is of total statement preparations, the command ID, the command text (the statement itself), and the source of the information (ttSQLCmdCacheInfo).
