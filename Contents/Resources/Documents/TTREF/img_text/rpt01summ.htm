Description of the illustration rpt01summ.gif

This graphic shows the Summary section of a ttStats report, including the following:

  • ID number and time of the beginning and ending snapshots

  • Information about memory usage and connections

    The begin value and end value are shown for connections statistics (such as connections.established.client_server and connections.established.direct) and for database memory size (such as db.size.perm_allocated.kb and db.size.perm_high_water_mark.kb) are shown.

  • Information about load profile

    The per second and per transaction rates are shown for items such as Ckpt Size, Log Bytes, and Log Reads.

  • Information about instance efficiency percentages

    Percentages are shown for Command Cache Hit, Non-Parse/Execs, Lock Hit, and Log Buffer No Wait.
