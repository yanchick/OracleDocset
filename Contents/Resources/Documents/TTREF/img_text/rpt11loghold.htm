Description of the illustration rpt11loghold.gif

This graphic shows information about log holds. For each checkpoint log hold LSN (loghold.Checkpoint_hold_lsn), it shows the begin value, end value, description (which checkpoint file), and source. For each replication log hold LSN (loghold.Replication_hold_lsn), it shows the begin value, end value, description (which replication subscriber), and source. In each case, the sources are shown to be the ttLogHolds and ttBookmark built-in procedures.
