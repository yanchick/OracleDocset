Description of the illustration careportsqlstats.jpg

This report page provides more detailed information about a single SQL statement executed during the Cache Advisor evaluation. It includes the execution count, and the projecte total reponse time for the Oracle database, the user-specified TimesTen Cache, and for a complete TimesTen Cache. It also provides the schema name, the SQL statement, and the cache groups associated with the statement.
