Description of the illustration dbr_replay_create_rep_nm.gif

The Replay Target section shows a Target Database read-only field pre-populated with "database." To the right of this is the Database Version number and Replay Host address.

The Task List table contains links to perform tasks.

For the Prepare Test Database task, the Set Up Test Database link and the Isolate Test Database link are provided.

For the Prepare for Replay task, the Preprocess Workload link and the Deploy Replay Clients link are provided.

For the Replay Workload on Test Database task, the Replay Workload link is provided.

The Workloads table shows entries for a single row for the following columns: Name, Database Name, Database Version, Database Time (hh:mm:ss), Workload Duration (hh:mm:ss), Storage Host, and Storage Location. The Workload Analyzer Report column has a clickable eyeglasses icon to access the report.

End of description.
