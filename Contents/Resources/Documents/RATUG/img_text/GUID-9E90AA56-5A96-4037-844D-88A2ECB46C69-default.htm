Description of the illustration spc_init_param.gif

The Initialization Parameters page is shown.

The Current tab is open.

The filter shows optimizer_mode in the Name field, All in the Basic field, All in the Modified field, All in the Dynamic field, and Optimizer in the Category field. The Go button for the filter has been clicked.

Below the filter is the initialization parameter table, which includes (from left to right) the Name, Help, Value, Comments, Type, Basic, Modified, Dynamic, and Category columns. The following values appear for the columns: optimizer_mode, an Information icon, FIRST_ROWS_100, no value, String, no value, no value, a check mark, and Optimizer.

The cursor is pointing to the Validate with SPA button at the top right of the graphic, above the Initialization Parameters page title.
