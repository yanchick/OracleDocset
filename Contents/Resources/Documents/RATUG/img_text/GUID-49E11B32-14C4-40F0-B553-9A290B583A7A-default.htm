Description of the illustration dbr_replay_compare_per_rep.gif

The title of the report is Compare Period Report: Consolidated Replay.

The table contains the following columns: Comparison Metric, HR, HR_CRM_SALES, CRM, HR_CRM_SALES, SALES, and HR_CRM_SALES. The Comparison Metric column contains the following rows: Total Sample Count, Total DB Time, Total CPU Time, Total WAIT Time, Total IO Time, IO Request Count, and Avg Time Per IO Request.

Data appears in all columns and rows.

This line of text appears at the bottom of the report: "This report compares the performance of multiple captures with the consolidated replay. However, the comparison is made with individual ASH capture data vs ASH replay data filtered for the specific captured workload. Time duration in this report is always represented in seconds."
