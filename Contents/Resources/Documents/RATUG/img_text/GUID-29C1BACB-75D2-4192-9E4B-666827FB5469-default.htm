Description of the illustration dbr_replay_tasks.gif

The Replay Task Summary section provides the Name, Description, and Owner fields.

The Captured Workloads table appears below this, and shows one row with entries for the following columns: Name, Database Name, Database Version, Database Time (hh:mm:ss), Workload Duration (hh:mm:ss), Storage Host, and Storage Location. The Workload Analyzer Report column shows a clickable eyeglasses icon to access the report.

The Replays table appears below the Captured Workloads table, and shows two rows with entries for the following columns: Name, Owner, Status, Database Name, Database Version, Creation Date. Neither row has an entry for the Description column. The following user controls appear above the table: View, Create, Delete, and Compare.

End of description.
