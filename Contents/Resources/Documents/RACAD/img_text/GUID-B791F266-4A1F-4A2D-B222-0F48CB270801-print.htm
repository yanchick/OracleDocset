Description of the illustration racad_vm_037.eps

This illustration shows a four-node cluster organized into two server pools—online and backoffice—plus the default free server pool. The online and backoffice server pools each contain two nodes. The values of the MIN_SIZE and MAX_SIZE server pool attributes are 2 and 3, respectively, for the online server pool, and the value of its IMPORTANCE server pool attribute is 5. The values of the MIN_SIZE and MAX_SIZE server pool attributes are 2 and 2, respectively, for the backoffice server pool, and the value of its IMPORTANCE server pool attribute is 10.

Two databases named dbsales and dbapps run in the online server pool, offering the browse, search, and salescart, and orderentry and billing services, respectively. Two databases named dberp and dbapps run in the backoffice server pool, offering the inventory and shipping, and erp and reports services, respectively.
