This is described in the accompanying text.

Use to change settings for when working with the auto update features.

  • Use HTTP Proxy

    Select to enter:

    • Host Name

      Specify the DNS name or the IP address of the database host.

    • Port Number

      Specify the port number.

    • Prox Server Requires Authentication

      Specify the user name and the password forthe proxy server if your proxy server requires authentication.

  • Test Proxy

    Use to test your proxy configuration.

  • Automatically Check for Updates

    Select to enable automatically checking for new DMU versions at startup.
