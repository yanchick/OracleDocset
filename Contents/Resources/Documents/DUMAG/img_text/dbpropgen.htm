Description of the illustration dbpropgen.gif

Displays:

  • Connection Name

    The name of the connection.

  • User Name

    The name of the user.

  • Role

    The name of the role.

  • Host Name

    The name of the host.

  • Port

    The port used.

  • Service Name

    The service name used.

  • Current Database Character Set

    The DMU checks your database and displays the current database character set.

  • Assumed Database Character Set

    Although the databse is set to a certain character set, it may actually contain data from a different one. As an example, if your database character set is WE8ISO8859P1 on the server and you have Windows clients, you may have mistakenly set the clients' NLS_LANG to the same as the database character set. In this case, you would have Windows code page data in the database, not what the server would expect.
