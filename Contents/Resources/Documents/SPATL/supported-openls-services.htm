14.1 Supported OpenLS Services

Spatial and Graph supports the following OGC OpenLS services:

  • Location Utility Service (geocoding)

  • Presentation Service (mapping)

  • Route Service (driving directions)

  • Directory Service (YP, or "Yellow Pages")

Spatial and Graph does not currently support the OGC OpenLS Gateway Service (mobile positioning).

For all supported services except Directory Service (YP, or Yellow Pages), you must first perform certain operations, which might included acquiring and loading third-party data, as well as configuring and deploying underlying technology on which the Spatial and Graph OpenLS service is based.
