Description of the illustration nine_inter.eps

This is a text description of a figure that illustrates the nine-intersection model, as described in text before the illustration. This illustration shows the bit mask (101001111) associated with two objects (A and B) that have a TOUCH relationship. The TOUCH relationship is defined in a list of relationship definitions after the figure.
