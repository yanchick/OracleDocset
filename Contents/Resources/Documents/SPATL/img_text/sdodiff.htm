Description of the illustration sdodiff.eps

This is a text description of **INTERNAL XREF ERROR**. It illustrates the result of an SDO_DIFFERENCE function operation, as described in text before the illustration. The returned polygon contains most of geom_1 (that is, geom_1 minus the part of geom_1 that is covered by geom_2).
