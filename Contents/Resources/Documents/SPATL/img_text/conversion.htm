Description of the illustration conversion.eps

This image illustrates converting a standard line string to an LRS line string, which causes a measure value (shown as underlined in the illustration) to be added in the measure dimension for each point. For the standard line string, the coordinates of the points shown are (0, 0), (10, 0), and (20, 0); for the LRS line string, the coordinates of the points shown are (0, 0, 0), (10, 0, 10), and (20, 0, 20). The underlined measure values of the LRS line string are 0, 10, and 20.
