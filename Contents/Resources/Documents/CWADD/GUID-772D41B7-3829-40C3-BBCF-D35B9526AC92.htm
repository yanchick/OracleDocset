cluvfy comp gpnp

Use the cluvfy comp gpnp component verification command to check the integrity of Grid Plug and Play on a list of nodes in a cluster.

Syntax

cluvfy comp gpnp [-n node_list] [-verbose]

Parameters

Table A-12 cluvfy comp gpnp Command Parameters

Parameter Description 
-n node_list

The comma-delimited list of nondomain qualified node names on which to conduct the verification.

If you do not specify this option, then CVU checks only the local node.

 
-verbose

CVU prints detailed output.
