cluvfy comp nodeapp

Use the component cluvfy comp nodeapp command to check for the existence of node applications, namely VIP, NETWORK, and ONS, on all of the specified nodes.

Syntax

cluvfy comp nodeapp [-n node_list] [-verbose]

Parameters

Table A-14 cluvfy comp nodeapp Command Parameters

Parameter Description 
-n node_list

The comma-delimited list of nondomain qualified node names on which to conduct the verification.

If you do not specify this option, then CVU checks only the local node.

 
-verbose

CVU prints detailed output.
