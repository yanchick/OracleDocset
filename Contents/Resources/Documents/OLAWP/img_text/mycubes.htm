Description of the illustration mycubes.gif

Results of the MyCubes <AWMNode> in aw.xml: This figure shows the navigation tree in Analytic Workspace Manager with the MyCubes folder selected. The folder is created by the <AWMNode> named MyCubes in the aw.xml document.

The report shows the query, select cube_name from user_cubes where aw_name = ?, and the result of the query, which is a column with the heading CUBE_NAME and the list of cubes, which are UNITS_CUBE and PRICE_CUBE.
