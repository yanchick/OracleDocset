Description of the illustration awmplugin.gif

Plugins Tab in the About Dialog Box: This screen capture shows the Plugins tab of the About dialog box. The tab contains a table with four columns. The column headings are Name, Version, Status, and Description. In the Name column are the names of the plug-ins, which are the following:

  • Cube Viewer Plug-in

  • Level Viewer Plug-in

  • Measure Viewer Plug-in

  • Delete Dimension Plug-in

  • Edit Dimension Plug-in

  • View XML Plug-in

The version of each plug-in is 1.0, except for the Edit Dimension Plug-in, for which the version number is 2.0. The status of each plug-in is loaded. The Description column contains the following:

  • Displays the name of a cube.

  • Displays the name of a level.

  • Displays the name of a measure.

  • Deletes a dimension in the MyDims folder.

  • Edits the short description of a dimension.

  • Displays the XML for an OLAP measure.
