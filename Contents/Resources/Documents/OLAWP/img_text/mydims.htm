Description of the illustration mydims.gif

Results of the MyDims <AWMNode> in aw.xml: This figure shows the navigation tree in Analytic Workspace Manager with the MyDims folder selected. The folder is created by the <AWMNode> named MyDims in the aw.xml document. The property inspector displays the report.

The report shows the query, select dimension_name, dimension_type from user_cube_dimensions where aw_name = ?, and the result of the query, which is a table with two columns. The left column has the heading DIMENSION_NAME. The rows of the column contain a list of dimensions, which are CHANNEL, CUSTOMER, PRODUCT, and TIME. The right column has the heading DIMENSION_TYPE. The rows of the column contain the type of the dimension. The values are STANDARD, STANDARD, STANDARD, and TIME.
