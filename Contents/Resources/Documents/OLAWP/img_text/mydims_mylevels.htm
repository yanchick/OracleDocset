Description of the illustration mydims_mylevels.gif

Results of the MyLevels <AWMNode> under the MyDims <AWMNode> in aw.xml: This figure shows the navigation tree in Analytic Workspace Manager with the MyLevels folder selected. The folder is created by the <AWMNode> named MyLevels in the aw.xml document. The property inspector displays the report.

The report shows the query, select level_name from user_cube_dim_levels where dimension_name = ?, and the result of the query, which is a column. The heading of the column is LEVEL_NAME. The rows of the column contain a list of levels, which are CHANNEL and TOTAL.
