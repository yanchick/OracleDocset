Description of the illustration mymeasrs.gif

Results of the MyMeasures <AWMNode> in cube.xml: This figure shows the navigation tree in Analytic Workspace Manager with the MyMeasures folder selected. The folder is created by an <AWMNode> in the cube.xml document. In the folder are the measures and calculated measures that are the result of the SQL statement in the <AWMNode>. The property inspector displays the SQL statement and a column that has the heading MEASURE_NAME. The column lists the measures and calculated measures of the cube.
