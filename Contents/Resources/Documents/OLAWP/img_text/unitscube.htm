Description of the illustration unitscube.gif

Results of the CubeViewerPlugin: This figure shows the navigation tree in Analytic Workspace Manager with the UNITS_CUBE cube selected in the MyCubes folder that is created by the aw.xml document. The property inspector displays the output of the CubeViewerPlugin, which is simply the name of the cube.
