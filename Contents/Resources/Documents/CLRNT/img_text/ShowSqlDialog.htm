Description of the illustration ShowSqlDialog.gif

This screen shot shows the Show Sql screen, which contains this code.

CREATE OR REPLACE LIBRARY CLRLIBRARY1_DLL AS '$ORACLE_HOME\
    bin\clr\CLRLibrary1_dll;
GRANT EXECUTE ON CLRLIBRARY1_DLL TO SCOTT;
GRANT EXECUTE ON DBMS_CLR TO SCOTT;
GRANT EXECUTE ON DBMS_CLRTYPE TO SCOTT;
GRANT EXECUTE ON DBMS_CLRPARAMTABLE TO SCOTT;
CREATE OR REPLACE FUNCTION SCOTT.GETDEPTNO(EMPNO NUMBER) RETURN BINARY_INTEGER;

Then there is an OK button and a Help button.
