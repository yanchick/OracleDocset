Description of the illustration usingDeploywizard1.gif

This window asks which data connection you want the wizard to use and provides a drop-down list of possible connections. The first item on the list is sys.Oracle.

There is a New Connection button.

There are buttons to go Back, Next, Finish, and Cancel.
