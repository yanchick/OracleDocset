XPointer Datatypes

Table 9-1 summarizes the datatypes of the XPointer package.

Datatype Description

typedef enum XppLocType {
      XPPLOC_TYPE_UNKNOWN = 0,

      XPPLOC_TYPE_BOOL    = 4,
      XPPLOC_TYPE_NUM     = 5,
      XPPLOC_TYPE_STR     = 6
    } XppLocType;
