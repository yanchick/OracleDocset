1.129 LDAP_DIRECTORY_SYSAUTH

Property Description

Parameter type

String

Syntax

LDAP_DIRECTORY_SYSAUTH = { yes | no }

Default value

no

Modifiable

No

Modifiable in a PDB

No

Basic

Yes

LDAP_DIRECTORY_SYSAUTH enables or disables directory-based authorization for SYSDBA and SYSOPER.
