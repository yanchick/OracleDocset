B.5 Java Scripts

The scripts in Table B-4 are useful only if the JServer option is installed.

Table B-4 Java Scripts

Script Name Description

initjvm.sql

Initializes JServer by installing core Java class libraries and Oracle-specific Java classes

rmjvm.sql

Removes all elements of the JServer

See Also: Oracle Database Java Developer's Guide

catjava.sql

Installs Java-related packages and classes
