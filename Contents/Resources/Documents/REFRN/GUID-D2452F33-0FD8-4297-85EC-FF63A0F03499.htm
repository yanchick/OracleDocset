1.97 FAST_START_MTTR_TARGET

Property Description

Parameter type

Integer

Default value

0

Modifiable

ALTER SYSTEM

Modifiable in a PDB

No

Range of values

0 to 3600 seconds

Basic

No

Oracle RAC

Multiple instances can have different values, and you can change the values at run time.

FAST_START_MTTR_TARGET enables you to specify the number of seconds the database takes to perform crash recovery of a single instance. When specified, FAST_START_MTTR_TARGET is overridden by LOG_CHECKPOINT_INTERVAL.
