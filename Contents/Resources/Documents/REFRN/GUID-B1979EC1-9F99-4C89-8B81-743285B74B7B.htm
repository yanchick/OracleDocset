4.115 DBA_CLUSTERING_TABLES

DBA_CLUSTERING_TABLES describes all the tables with an attribute clustering clause. Its columns are the same as those in ALL_CLUSTERING_TABLES.

Note:

This view is available starting with Oracle Database 12c Release 1 (12.1.0.2).
