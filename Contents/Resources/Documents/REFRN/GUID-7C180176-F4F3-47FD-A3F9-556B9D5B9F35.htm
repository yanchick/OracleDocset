6.159 DBA_ZONEMAP_MEASURES

DBA_ZONEMAP_MEASURES describes the measures for all the zone maps in the database. Its columns are the same as those in ALL_ZONEMAP_MEASURES.

Note:

This view is available starting with Oracle Database 12c Release 1 (12.1.0.2), and is intended for use with Oracle Exadata release 12.1.2.1.1 or later.
