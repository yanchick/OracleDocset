Description of the illustration syntax_2.eps

This is a text description of syntax2.gif. This image shows a syntax diagram, which shows the following available options for 4-BIT and 8-BIT operator vlaues:

  • LUT RGB

  • LUT RGB A

  • LUT RGB T

  • LUT GRAY

  • LUT GRAY SCALE

  • LUT GRAY A

  • LUT GRAY T

  • LUT GRAY SCALE A

  • LUT GRAY SCALE T

In the preceding list, the spelling G-R-A-Y can substituted with the spelling G-R-E-Y.
