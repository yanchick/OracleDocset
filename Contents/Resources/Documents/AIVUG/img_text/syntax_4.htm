Description of the illustration syntax_4.eps

This is a text description of syntax4.gif. This image shows a syntax diagram, which shows the following available options for 24-BIT and 32-BIT operator values:

  • 24 BIT

    • RGB

    • BIP RGB

    • BIL RGB

    • BSQ RGB

    • BIP DRCT RGB

    • BIL DRCT RGB

    • BSQ DRCT RGB

  • 32 BIT

    • DRCT RGB A

    • RGB A
