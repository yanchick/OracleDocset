Description of the illustration exp_create_user_role_privs.gif

This is a screenshot of the Privileges page. The available system privileges and roles table is on the left side of the page. The selected system privileges and roles table is on the right side of the page. The right arrow button and left arrow button that appear between the two tables are used to move selected system privileges and roles between the two tables. The APPDEV role and the CREATE SESSION system privilege have been moved to the selected system privileges and roles table.
