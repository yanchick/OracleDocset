Description of the illustration row6_sqlta_tun_recs.gif

This is a screen shot of the Recommendation Details page in EM Express. The top left side of the page includes the page name, and to the far right the Page Refreshed field shows when the page was last refreshed.

On the left below the page name is the Alternative Plan(s) section. It describes the performance recommendation and provides an overview of the recommendation.

Below the Alternative Plan(s) section is the Compare Explain Plans section. This section includes the Create SQL Plan Baseline button. The Plan Hash Value row appears below the Create SQL Plan Baseline button. The Plan Hash Value row includes these columns and values:

  • Plan Hash Value: 770275272

  • Last Seen: 2012-10-23/10:08:34

  • Elapsed Time: 19.1s

  • Origin: Cursor Cache

  • Note: This column is blank

On the Original Plan tab (from left to right) are the Graphical and Tabular buttons. The Tabular button is selected.

Below the Graphical and Tabular buttons is a table that shows the steps in the original explain plan for the selected SQL statement.

The following columns exist in the table:

  • Operation

  • Object

  • Predicate

  • Pruning

  • Operation Cost

  • Estimated Rows

  • Estimated Bytes
