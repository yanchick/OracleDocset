Description of the illustration man_mem_mgmt_crop.gif

This is a screenshot that shows the Memory Settings section of the Memory Management page in EM Express when Manual Shared Memory Management is enabled. The Memory Settings section includes the SGA Memory subheading and the PGA Memory subheading. Under the subheading SGA Memory, several fields and values appear. The value for the Memory Management field is Manual; the value of the db_cache_size field is 15MB, the value of the java_pool_size field is 88 MB, and the value of the shared_pool_size field is 229MB. Under the subheading PGA Memory, the value for the Management Mode field is Auto and the value of the pga_aggregate_target field is 28MB.
