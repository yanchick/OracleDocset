Description of the illustration buffer_cache_advisor_graph.gif

This is a screenshot of the Buffer Cache Advisor graph on the Memory Management page in EM Express. The Buffer Cache Advisor graph appears on the Memory Management page when manual shared memory management mode is enabled for the database. The horizontal axis of the graph is labeled db_cache_size (MB) and shows possible values for the DB_CACHE_SIZE initialization parameter. The horizontal axis starts with a value of 0 and displays values for the DB_CACHE_SIZE initialization parameter in 100 MB increments up to 400 MB. The vertical axis of the graph is labeled Estimated Percentage of Reads Saved. The vertical axis starts with a value of -700 and decreases in increments to 0%. An orange line on the graph plots possible values for the DB_CACHE_SIZE initialization parameter. The current setting for the DB_CACHE_SIZE initialization parameter is shown by a blue dot near the 600 MB value.
