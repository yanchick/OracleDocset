Description of the illustration sqld_create_index_new.gif

This screenshot shows the Create Index dialog box in SQL Developer, with the Definition tab displayed. The top of the page has the Schema field and Name field. The Schema field has the value SH. The Name field has the value PROD_DESC_IDX. On the Definition tab, the Schema field has a value of SH, and the Table field has a value of PRODUCTS. The Index Type field has the value Non-Unique. In the Expression section, the PROD_DESC appears in the Expression column, and Not Specified appears in the Order column.
