Description of the illustration current_findings.gif

This screen shot shows the Performance Hub: Real Time - Last Hour page with the Current ADDM Findings tab selected.

The Current ADDM Findings tab in the Performance Hub shows any ADDM findings for the past 5 minutes in real time.

Findings are listed in the Findings table, which includes the Finding, Scope, Priority, Number of Recommendations, and Impact columns.

Details about findings are listed in the Finding Details section below the Findings table.
