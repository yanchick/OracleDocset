Description of the illustration sqld_create_table_new.gif

This screenshot shows the Create Table dialog box in SQL Developer, with the Table tab displayed. In the Schema field, HR has been selected. In the Name field, PURCHASE_ORDERS has been entered. The Table tab includes the columns PK, Name, Data Type, Size, Not Null, Default, and Comment. The information for the columns in the PURCHASE_ORDERS table has been entered in the dialog box, as specified in the table in the introduction to this section.
