Description of the illustration sql_tuning_settings_dibox.gif

This is a screen shot of the SQL Tuning Settings dialog box. The Automatic SQL Tuning field has a check mark after it, indicating that Automatic SQL Tuning Advisor is enabled. The Time Limit per Statement (seconds) field has a value of 1200. The Automatic Implementation of SQL Profiles option is disabled. The Maximum SQL Profiles Implemented per Execution option has a value of 20. The Maximum SQL Profiles Implemented (Overall) option has a value of 10000.
