Description of the illustration sqld_view_table_data.gif

This screenshot shows SQL Developer displaying the data for a table. The left side of the screenshot shows the Connections navigator, with the Tables node open, and with several table names displayed. The COUNTRIES table is selected. The right side of the screenshot shows the object pane, with the COUNTRIES tab appearing, and the Data subtab displayed. The Data subtab displays the data for the COUNTRIES table.
