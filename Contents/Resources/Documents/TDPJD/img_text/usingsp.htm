Description of the illustration usingsp.gif

This image shows the insert_emp.jsp page as shown in the browser with the options to insert an employee data using JDBC or through a SQL stored procedure. It also contains a table with form input controls and associated labels. There is also an Add Employee button that submits these values when the employee details are specified.
