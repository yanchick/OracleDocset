Description of the illustration jspstatcont.gif

This image shows the Java Source Editor window in JDeveloper. For static data, the Java Source Editor acts as a WYSIWYG (What You See Is What You Get) editor. It shows text formatted with the applied styles, as it would appear in the web page itself.
