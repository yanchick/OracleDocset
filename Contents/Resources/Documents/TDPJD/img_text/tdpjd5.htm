Description of the illustration tdpjd5.gif

Figure 3-5 shows the Create Java Class dialog box. It accepts user input to create a Java class. In this dialog box, you specify a name for the Java class and the package to which it must belong. You also select the class or interface that it extends from a dropdown list. You can also specify other options from the Optional Attributes panel.
